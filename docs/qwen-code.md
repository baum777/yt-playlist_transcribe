# Qwen Code

Qwen Code ist ein optionaler Terminal-Agent für Repo- und Coding-Workflows.

## Installation

```bash
npm install -g @qwen-code/qwen-code@latest
```

## Start

```bash
qwen
```

## Hinweis

Für headless- oder CI-Workflows ist ein API-Key-basierter Zugang meist sinnvoller als OAuth.

Diese Dokumentation ist rein optional und wird nicht als Laufzeitabhängigkeit der Website verwendet.
